Gentoo LinuxのEee PC 900へのインストール: 日本語入力システムのインストール

(工事中 90%)

はじめに

uim上にTUT-Codeを乗せる。

uimのインストール

% emerge -pv uim

/etc/portage/make.confのCPU_FLUGS_X86にsseを追加した。

% sudo emerge -av uim

31パッケージがインストールされるのですこし時間がかかるかもしれない。 1時間20分かかった。

.xinitrcの編集

日本語入力を可能にするためにはいくつかの環境変数の設定が必要になる。 またuim-ximに--engine=tutcodeオプションをつけて立ち上げる。 これはXを立ち上げるときに行うので.xinitrcに書き込む。

% vi ~/.xinitrc
export LANG="ja_JP.UTF-8"
export XMODIFIERS="@im=uim"
uim-xim --engine=tutcode&
exec xmonad

一度Xから抜けて再度入る。 Shift-Spaceで日本語の入力が可能となる。

交ぜ書き変換用の辞書

ホームディレクトリに交ぜ書き含換用の辞書を置いておく。 ここにとりあえずの.mazegaki.dicを置く。

以下のようにしてファイルを取得する。

% curl -o ~/.mazegaki.dic https://skami.iocikun.jp/computer/genmtoo/install/eeepc_900/mazegaki.dic

Xに入り直すと交ぜ書き変換が可能となる。

成功

これで日本語入力が可能となった。

追加の設定

現在の入力モードがわからないと面倒なので追加の設定をする。 またEscキーで日本語入力も抜けられるとvimで使う場合に便利。

% uim-pref-gtk

「カーソルの側に入力モードを表示」をチェックする。 「TUT-Code」のページに移動し「vi協調モードを有効にする」をチェックする。
