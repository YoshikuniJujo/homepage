Gentoo LinuxのEee PC 900へのインストール: Portage樹のインストール

portageのスナップショット

emerge-webrsyncでportageのスナップショットをインストール。

# emerge-webrsync

portage樹の全体をダウンロードして配置するので多少時間がかかる。 僕の環境では10分かかった。

portage樹の更新

emerge-webrsyncはたいていは24時間以内のスナップショットをインストールする。 更新の必要はない。 インストールがうまくいっていることの確認としてportage樹を更新してみる。

# emerge --sync

2分ほどかかる。 RMSの「フリーソフトウェアと自由な社会」でも読んで時間をつぶそう。

ニュース

portage樹を更新したときにニュースがとどく場合がある。 これはeselect news list, eselect news read [番号]によって読める。 またeslect news purgeで読んだニュースを削除できる。

時々重要なニュースがあったりする。 「/usrを別パーティションにしている場合initramfsを作らないと起動しなくなる」とか。

プロファイルを選択する

様々な設定がプロファイルにまとめられている。 まずはデフォルトの状態でのプロファイルを確認してみよう。

eselect profile list

デフォルトのプロファイルはdefault/linux/x86/13.0である。 3番目にdefault/linux/x86/13.0/desktopがある。 デスクトップの用途なのでこのプロファイルにしておく。

eselect profile set 3
