Gentoo LinuxのEee PC 900へのインストール: Portage樹のインストール

(工事中 90%)

portageのスナップショットのインストール

emerge-webrsyncを使ってportageのスナップショットをインストールする。

# emerge-webrsync

portage樹の更新

emerge-webrsyncはたいていは24時間以内のスナップショットをインストールするので、 特に更新の必要はない。 ここではインストールが上手くいっていることの確認として行うこととする。

# emerge --sync

ニュース

portage樹を更新したときにニュースがとどく場合がある。 これはeselect news list, eselect news readによって読むことができる。 またeslect news purgeでニュースを削除できる。

プロファイルを選択する

様々な設定がプロファイルにまとめられている。 まずはデフォルトの状態でのプロファイルを確認してみよう。

eselect profile list

現在のプロファイルはdefault/linux/x86/13.0であることがわかる。 また3番目にdefault/linux/x86/13.0/desktopがある。 今回はデスクトップの用途なので一応3番目のプロファイルにしておくことにする。

eselect profile set 3
