fstabの作成

ブート時にマウントするパーティションを指定する等のために /etc/fstabファイルを作成する必要がある。

今回のパーティション構成

  • / /dev/sda1
  • /usr/ /dev/sdb1
  • swap /dev/sda2

fstabファイルの6つのフィールド

  1. パーティションのデバイスファイルへのパス
  2. マウントポイント
  3. ファイルシステムの種類
  4. マウントオプション
  5. ダンプが必要かどうか。普通は0にしておけば良い
  6. fsckがチェックする順番を決める
    • /は1とする
    • 他はチェックが必要なパーティションは2とし
    • チェックが不要なパーティションは0とする

fstabファイルの作成

もともと用意されている/etc/fstabを編集する。

# vi /etc/fstab

以下のような内容にする。

/dev/sda1 / ext4 noatime 0 1
/dev/sdb1 /usr ext4 noatime 0 2
/dev/sda2 none swap sw 0 0
/dev/cdrom /mnt/cdrom auto noauto,ro 0 0

ネットワークの設定

ホスト名とドメイン情報

PCの名前を決める。この名前は後から変えることができる。 今回のPCには黒猫のシールがはってあるのでblack_catとしておく。

vi /etc/conf.d/hostname
hostname="black_cat"

今回はドメイン名を指定する必要はない。 ログイン時にhoge.(none)と表示されないように、以下のようにしておく。

vi /etc/issue
('.\O'を削除)

ネットワークへの接続の設定

最終的には有線LANの他にテザリングでも接続できるようにする。 余力があれば無線LANの設定もしたい。 しかし、現段階では有線LANのみを扱う。

netifrcをインストールする。 (このパッケージは比較的新しいようで僕は初めて見る)

# emerge --ask --noreplace net-misc/netifrc

今回はすでにインストールされていたようだ。 DHCPクライアントは後でインストールする。

有線LANに関してはDHCPを使用することにする。 まずは有線LANのインターフェース名を調べよう。

# ifconfig | less

ここで表示されるインターフェースのうちenp3s0が有線LANのインターフェースのようだ。

# vi /etc/conf.d/net
config_enp3s0="dhcp"

/etc/init.d/net.enp3s0を作成する。

# cd /etc/init.d
# ln -s net.lo net.enp3s0

ブート時に自動でネットワークをつなげるためには以下のようにする。

# rc-update add net.enp3s0 default

しかし、今回はブート後に明示的に /etc/init.d/net.enp3s0 startのようにすることにする。

カーネルに有線LANのドライバを追加

Eee PC 900のLANカードを認識させるためのドライバーを組み込むために、 カーネルに追加の設定が必要となる。

# lspci | grep -i ethernet
01:00.0 Ethernet controller: Qualcomm Atheros AR242x / AR542x Wireless Network Adapter (PCI-Express) (rev 01)
03:00.0 Ethernet controller: Qualcomm Atheros Attansic L2 Fast Ethernet (rev a0)

有線LANはQualcomm Atheros Attansic L2 Fast Ethernetという製品であるようだ。 このカードのドライバを追加することにする。

# cd /usr/src/linux
# make menuconfig
Device Driver --->
[*] Network device support --->
[*] Ethernet driver support --->
[*] Atheros devices
<*> Atheros L2 Fast Ethernet support

カーネルの再コンパイルとインストール

# make && make modules_install
# make install

念のためinitramfsも作り直しておく。

# genkernel --install initramfs
# ls /boot/initramfs*
