Gentoo LinuxのEee PC 900へのインストール: タイムゾーンとロケールの設定

タイムゾーンの設定

タイムゾーンを設定する。何があるかを調べる。

# ls /usr/share/zoneinfo

上記の結果からタイムゾーンを選び/etc/timezoneに記述。

# echo "Japan" > /etc/timezone

timezone-dateパッケージの再設定。

# emerge --config sys-libs/timezone-data

ロケールの設定

# vi /etc/locale.gen

アメリカ英語と日本語の値だけをデコメント。

# locale-gen
# locale -a

システム全体で使うロケールを選択する。

# eselect locale list
# eselect locale set 9

# cat /etc/env.d/02locale

# env-update && source /etc/profile
