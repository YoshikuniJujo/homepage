Gentoo LinuxのEee PC 900へのインストール: スカイプのインストール

(工事中 80%)

skypeパッケージのインストール

% sudo emerge -av skype

ACCEPT_KEYWORDSとLICENSEへの変更がある。

% sudo etc-update
% sudo emerge -av skype

ある程度時間がかかりそうだ。6時間くらいかかる。

失敗

おそらくpulseaudioあたりの問題かと思われる。 alsaは使えているし、とくにSkypeを使う予定もないので「まあいいや」

成功

dbusが走っていなかったのが問題だったようだ。

% sudo /etc/init.d/dbus start
% pulseaudio --start

dbusを起動時に走らせる

% sudo rc-update add dbus default

再起動してチェックしてみる。
