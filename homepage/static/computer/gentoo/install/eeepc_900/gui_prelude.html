Gentoo LinuxのEee PC 900へのインストール: GUI: はじめに

マウスレスへ

GUIをインストールし設定していく。 タイル型ウィンドウマネージャxmonadを採用する。 タイル型では基本的にウィンドウ同士を重ねない。 画面を最も効率的に使用する方法だ。 また、マウスとキーボードを比較すると、 熟練した場合の作業効率はキーボードのほうが断然高い。 xmonadではマウスが必要な場面はすくなく、多くの作業をキーボードのみで行える。
