はじめに

ウィンドウマネージャとしてxmonadを使用する。 xmonadはHaskellで書かれていて、コンパイルのためにGHCが必要になる。

GHCのインストール

GHCのインストールには非常に時間がかかる。 すくなくとも以前はとても時間がかかった。 覚悟してemergeする。 まずはUSEフラグの確認から。

% emerge -pv ghc

よくわからないがllvmを有効にしておくことにする。

% sudoedit /etc/portage/make.conf
...
USE="bindist \
...
zsh-complition clang llvm"
...

% sudo emerge -av ghc

以前は10時間ほどかかったこともあったが、最近はかなり速くなっている。 それでもそれなりの時間はかかると思われるので、 お茶でも飲んで待つことにする。
