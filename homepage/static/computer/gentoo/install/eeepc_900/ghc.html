Gentoo LinuxのEee PCへのインストール: GHCのインストール

(工事中 90%)

はじめに

ウィンドウマネージャとしてxmonadを使用する。 xmonadはHaskellで書かれている。 コンパイルのためにGHCが必要だ。

GHCのインストール

GHCのインストールには非常に時間がかかる。 すくなくとも以前はとても時間がかかった。 覚悟してemergeする。 まずはUSEフラグの確認。

% emerge -pv ghc

よくわからないがllvmを有効にしておくことにする。

% sudoedit /etc/portage/make.conf
...
USE="bindist \
...
zsh-complition clang v4l -qt4 llvm"
...

USE変数を変更したので一応システムを更新しておく。

% sudo emerge -avu --deep --newuse @world

とくに更新はないようだ。GHCをインストールする。

% sudo emerge -av ghc

お茶でも飲んで待つことにする。 4時間半かかる。
