Gentoo LinuxのEee PC 900へのインストール: vimのインストール

はじめに

いろいろな設定作業においてnanoを使うのがめんどくさい。 vimをインストールしておく。

vimのインストール

vimとともに何がインストールされるかのチェック。

# emerge -pv vim

37個のパッケージがインストールされる。 vimだけあれば良いのでとりあえずUSE変数Xを無効にしてみる。

# USE="-X" emerge -pv vim

こうすると20個に減る。 USE変数rubyを無効にしてみる。

# USE="-X -ruby" emerge -pv vim

さらに9個に減る。この設定でインストールする。

# USE="-X -ruby" emerge -av vim

15分ほど時間がかかる。 昼食とした。
