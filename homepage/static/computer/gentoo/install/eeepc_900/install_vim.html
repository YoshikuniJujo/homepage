Gentoo LinuxのEee PC 900へのインストール: vimのインストール

(工事中 90%)

はじめに

いろいろな設定作業においてnanoを使うのがめんどくさいのでここでvimをインストール してしまおう。

vimのインストール

まずはvimとともに何がインストールされるかをチェックする。

# emerge -pv vim

37個のパッケージがインストールされるようだ。 今はvimだけあれば良いのでUSEから変数をいくつか無効にして試みる。

# USE="-X" emerge -pv vim

こうすると20個に減る。

# USE="-X -ruby" emerge -pv vim

さらに9個に減る。 だいたい良さそうなのでこの設定でインストールすることにする。

# USE="-X -ruby" emerge -av vim

お茶でも飲みながら待つ。
