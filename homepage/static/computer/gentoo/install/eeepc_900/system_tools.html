システムツールのインストール

(工事中 90%)

システムロガー

Gentooではすくなくとも3つからロガーを選択することになる。 sysklogd, syslog-ng, metalogである。 あまりよくわからないが、metalogが良さそうな気がする。

# emerge --ask app-admin/metalog
# rc-update add metalog default

クロンデーモン

同様にクロンデーモンにもいくつかの選択肢がある。 これもなんとなくfcronを選択。

# emerge --ask sys-process/fcron
# rc-update add fcron default
# crontab /etc/crontab

ファイルインデックス

mlocateをとりあえずインストールしておく。

# emerge --ask sys-apps/mlocate

ファイルシステムツール

ファイルシステムにはext4のみを使っている。 e2fsprogsはすでにインストールされているのでここで新たにインストールする必要はない。

DHCPクライアント

ネットワークの設定にはDHCPを使用するのでDHCPクライアントが必要となる。

# emerge --ask net-misc/dhcpcd
