システムツールのインストール

システムロガー

Gentooではすくなくとも3つからロガーを選択することになる。 sysklogd, syslog-ng, metalogである。 あまりよくわからないが、metalogが良さそうだ。

# emerge --ask --verbose app-admin/metalog

2分ほどかかる。

# rc-update add metalog default

クロンデーモン

同様にクロンデーモンにもいくつかの選択肢がある。 これもなんとなくfcronを選択。

# emerge --ask --verbose sys-process/fcron

11パッケージがインストールされる。 30分ほどかかる。

# rc-update add fcron default
# crontab /etc/crontab

ファイルインデックス

mlocateをとりあえずインストールしておく。1分ほどかかる。

# emerge --ask --verbose sys-apps/mlocate

ファイルシステムツール

ファイルシステムにはext4のみを使っている。 ext4を扱うツールを集めたパッケージe2fsprogsはすでにインストールされている。

DHCPクライアント

ネットワークの設定にDHCPを使用するのでDHCPクライアントが必要だ。 1分ほどかかる。

# emerge --ask --verbose net-misc/dhcpcd
