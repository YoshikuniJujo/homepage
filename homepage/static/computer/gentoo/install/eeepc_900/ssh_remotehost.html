Gentoo LinuxのEee PC 900へのインストール: SSHによるリモートホストの操作

(工事中 60%)

はじめに

Eee PCからもリモート端末が操作したいので。

メインマシンからの秘密鍵、公開鍵をコピー

メインマシンの秘密鍵、公開鍵をコピーする。

% cp .ssh/id_rsa [USBメモリ等]
% cp .ssh/id_rsa.pub [USBメモリ等]

Eee PC 900に秘密鍵、公開鍵をコピーする。

% cp [USBメモリ等]/id_rsa ~/.ssh
% cp [USBメモリ等]/id_rsa.pub ~/.ssh

SSHによる接続

ssh -p [ポート番号] [ユーザ名]@[ホスト名]
