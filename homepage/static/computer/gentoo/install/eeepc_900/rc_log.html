Gentoo GNU/LinuxのEee PC 900へのインストール: rcのログを記録

(工事中 90%)

はじめに

デーモンの開始に失敗した場合、画面にはエラーメッセージが表示されるが、 デフォルトの設定ではログには残らない。 これをログに記録しあとから読めるようにする。

rc_loggerをYESに

% sudoedit /etc/rc.conf
...
rc_logger="YES"
...

試してみる

再起動する。 exitを何回かしてターミナルを終了し、 Ctrl-Alt-qでXを終了したうえで

% sudo reboot

% cat /var/log/rc.log
