Gentoo LinuxのEee PC 900へのインストール: HostAPでアクセスポイントとする

(工事中 0%)

はじめに

Eee PC 900はWi-Fiカードを内蔵している。 Gentooからそれを使うことができる。 しかし、それをテストするにはアクセスポイントが必要である。 そのために、まずはアクセスポイントを構築することにする。 アクセスポイント自体のテストにはスマホを使うことにする。

使用するデバイス

USB無線LANアダプター: SONY UWA-BR100
