Gentoo LinuxのEee PC 900へのインストール: インストールCDからのブート

(工事中 90%)

Eee PC本体の用意

  1. 電源を接続する
  2. 有線LANを接続する
  3. キーボードを接続する
  4. USB CD(DVD)-ROMドライバを接続する
    1. インストールCDを挿入する

ブート

  1. 電源を入れる
  2. 「boot: 」という表示が出たらキーボードからリターンキーを入力する
    • キーを押さないで15秒たつとディスクからブートしてしまう
  3. ルートでログインしたコンソールが表示される
    • livecd ~ #

ネットワーク接続の確認

DHCPが適切に動いている環境であればこの時点でネットに接続できているはずだ。 pingで確認してみよう。

# ping gentoo.org -c 3

64 byte from www.gentoo.org ...
