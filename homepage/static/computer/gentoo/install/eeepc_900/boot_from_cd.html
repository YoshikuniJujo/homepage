Gentoo LinuxのEee PC 900へのインストール: インストールCDからのブート

Eee PC本体の用意

  1. 電源を接続する
  2. 有線LANを接続する
  3. キーボードを接続する
  4. USB CD(DVD)-ROMドライバを接続する
  5. インストールCDを挿入する

ブート

  1. 電源ボタンを押す
  2. 「boot: 」という表示が出たらキーボードからリターンキーを入力
    • キーを押さないと15秒でディスクからブートしてしまう
  3. コンソールが表示される
    • livecd ~ #

x86_64でないことの確認

# lscpu
...
CPU op-mode(s): 32-bit
...

x86_64であればCPU op-mode(s): 32-bit, 64-bitとなる。

ネットワーク接続の確認

DHCPが適切に動いていればすでにネットに接続されているはずだ。 pingで確認する。

# ping gentoo.org -c 3
PING gentoo.org ...
64 byte from www.gentoo.org ...
64 byte from www.gentoo.org ...
64 byte from www.gentoo.org ...

--- gentoo.org ping statistics ---
3 packets transmitted, 3 recieved, ...

ちゃんと接続されているようだ。
