Gentoo LinuxのEee PC 900へのインストール: ブートローダの設定

ブートローダについて

BIOSを使っている場合にはGRUB2またはLILOを使い、 UEFIを使っている場合にはefibootmgrを使う。 今回はGRUB2を使うことにする。

GRUB2

まずはgrubパッケージをインストールする。

# emerge --ask sys-boot/grub

# grub2-install /dev/sda

# grub2-mkconfig -o /boot/grub/grub.cfg

再起動

さて、緊張の瞬間。 10回に1回くらいはうまくいかないものだ。 覚悟を決めて再起動する。

# exit
# cd
# umount -l /mnt/gentoo/dev{/shm,/pts,}
# umount /mnt/gentoo{/boot,/sys,/proc,}
# reboot

CDは出しておこう。

大失敗

原因はだいたいわかった。 はじめのパーティションを切るところで grub用のパーティションを作成する必要があったらしい。 パーティションの切りかたのミスなので、 ほど最初からやりなおすことになる。
