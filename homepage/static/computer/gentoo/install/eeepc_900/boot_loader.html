Gentoo LinuxのEee PC 900へのインストール: ブートローダの設定

ブートローダについて

使っているマシンに BIOSが使われている場合にはGRUB2またはLILOを使い、 UEFIが使われている場合にはefibootmgrを使う。 Eee PC 900にはBIOSが使われている。 GRUB2を使うことにする。

GRUB2

grubパッケージのインストール。

# emerge --ask --verbose sys-boot/grub

20分ほどかかる。

# grub2-install /dev/sda

# grub2-mkconfig -o /boot/grub/grub.cfg

再起動

さて、緊張の瞬間。 10回に1回くらいはうまくいかないものだ。 覚悟を決めて再起動する。

# exit
# cd
# umount -l /mnt/gentoo/dev
# umount /mnt/gentoo/usr
# umount /mnt/gentoo/proc
# umount /mnt/gentoo/sys
# umount /mnt/gentoo # reboot

/mnt/gentoo/sysがumountできない。 シャットダウンしてしまおう。

CDは出しておこう。

成功

grub用のパーティションを作成しておいたのでうまくいった。 プロンプトが出る。

yellow_keys login:

rootと入力しリターンを押す。 設定したroot用のパスワードとリターンを入力。

yellow_keys login: root
Password:
yellow_keys ~ #
