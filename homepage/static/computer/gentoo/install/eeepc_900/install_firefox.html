Gentoo LinuxのEee PC 900へのインストール: Firefoxのインストール

はじめに

Webを参考にインストール作業を行っている。 Firefoxを入れておけばEee PC 900単体でのインストールができる。

firefoxのインストール

% emerge -pv firefox

とくに問題なさそうだ。

% sudo emerge -av firefox

9時間強かかる。

vimperatorの導入

拡張機能vimperatorを導入。 .vimperatorrcの作成。

% vi ~/.vimperatorrc
set noinsertmode
