Gentoo LinuxのEee PC 900へのインストール: Firefoxのインストール

(工事中 90%)

はじめに

Webを参考にインストール作業を行っているので、 Firefoxを入れておけばEee PC 900単体でのインストールも可能となる。 よってここでFirefoxを入れておく。

syncとアップデート

Firefoxのインストールにもかなりの時間がかかるので、 せっかくならば最新版を入れておきたい。 よって先にsyncをする。 syncしたらまずは既存のシステムを最新にする。

また、忘れていたが循環依存の問題のために一時的に無効にしていたUSE変数を もとにもどしておくことにする。 また、suspendパッケージに必要だった追加のUSEがxmltoファイルに入っている。 suspendパッケージは使わなかったのでこれもついでに消してしまおう。

% sudo rm /etc/portage/package.use/xmlto
% sudo rm /etc/portage/package.use/qtgui

Portage樹の更新。

% sudo emerge --sync

3分ほどかかる。 システムのアップデート。

% sudo emerge -avu --deep --newuse system

USEの変更を行った部分で4つのパッケージが再コンパイルされる。 1時間20分かかった。

次はシステムに含まれないパッケージのアップデートを行う。

% emerge -pvu --deep --newuse world

古いパッケージはとくにないようだ。

firefoxのインストール

% emerge -pv firefox

とくに問題なさそうだ。

% sudo emerge -av firefox

9時間かかった。

vimperatorの導入

firefox内で拡張機能vimperatorを導入する。 .vimperatorrcの作成。

% vi ~/.vimperatorrc
set noinsertmode
