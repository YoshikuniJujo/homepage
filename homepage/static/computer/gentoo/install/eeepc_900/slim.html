Gentoo LinuxのEee PC 900へのインストール: SLiMのインストールと設定

SLiMのインストール

% sudo emerge -av x11-misc/slim

起動時に走らせる

% sudoedit /etc/conf.d/xdm
DISPLAYMANAGER="slim"
% sudo rc-update add xdm default

.xinitrcを使うようにする

% sudoedit /etc/slim.conf

再起動

再起動してみる。
