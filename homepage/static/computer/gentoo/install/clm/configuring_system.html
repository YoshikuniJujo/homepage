fstab

/etc/fstabを編集する。 以下のような構成にする。

  • /dev/sda3 /
  • /dev/sda2 /boot
  • /dev/vg/usr /usr
  • /dev/vg/home /home
  • /dev/vg/opt /opt
  • /dev/vg/var /var
  • /dev/vg/tmp /tmp
  • /dev/sda4 swap

# nano /etc/fstab
/dev/sda3 / ext4 noatime 0 1
/dev/sda2 /boot ext4 noatime 0 2
/dev/vg/usr /usr ext4 noatime 0 2
/dev/vg/home /home ext4 noatime 0 2
/dev/vg/opt /opt ext4 noatime 0 2
/dev/vg/var /var ext4 noatime 0 2
/dev/vg/tmp /tmp ext4 noatime 0 2
/dev/sda4 none swap sw 0 0
/dev/cdrom /mnt/cdrom auto noauto,ro 0 0

hostname

ホスト名はとりあえずclmとしておくことにする。

# nano -w /etc/conf.d/hostname
...
hostname="clm"

一応ドメイン名も設定しておくことにする。

# nano -w /etc/conf.d/net
...
dns_domain_lo="local.hogehoge.jp"

hosts

/etc/hostsファイルに自分の名前を設定しておく。

# nano -w /etc/hosts
...
127.0.0.1 clm clm.local.hogehoge.jp localhost
