ISOイメージのダウンロード

ミラーリストから ミラーサイトを選択し以下のようにディレクトリをたどる。

  1. releases/
  2. amd64/
  3. autobuilds/
  4. current-install-amd64-minimal/

install-amd64-minimal-2015XXXX.isoと install-amd64-minimal-2015XXXX.iso.DIGESTS.ascをダウンロードする。

ファイルの完全性の確認

% gpg --recv-keys 0xBB572E0E2D182910
% gpg --verify install-amd64-minimal-2015XXXX.iso.DIGESTS.asc
...
主鍵のフィンガー・プリント: 13EB BDBE DE7A 1277 5DFD B1BA BB57 2E0E 2910

フィンガー・プリントをGentoo署名ページで確認する。

% grep -A 1 -i sha512 install-amd64-minimal-2015XXXX.iso.DIGESTS.asc
# SHA512 HASH
d3e4d5b7... install-amd64-minimal-2015XXXX.iso
# SHA512 HASH
...
% sha512sum install-amd64-minimal-2015XXXX.iso
d3e4d5b7...

CDに焼く

% cdrecord install-amd64-minimal-2015XXXX.iso

確認。

% sudo mount /dev/cdrom /mnt/free/
% ls /mnt/free/
README.txt gentoo.efimg gentoo.efimg.mountPoint image.squashfs isolinux livecd
% sudo eject

ラベルを書く。

Gentoo GNU/Linux install-amd64-minimal-20150618.iso sha512sum: d3e4d5b7da51a316db0cb0db ... 535d9a9ebea4aab32a
