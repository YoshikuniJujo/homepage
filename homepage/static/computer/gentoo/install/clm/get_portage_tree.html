Gentoo GNU/Linuxの自当機(CLM)へのインストール: Portage樹の取得と更新

(工事中 0%)

Portage樹の取得

# emerge-webrsync

Portage樹の更新

# emerge --sync

プロファイルの選択

# eselect profile list
# eselect profile set 3

tmuxの導入

# emerge -av tmux

vimの導入

# emerge -av vim
