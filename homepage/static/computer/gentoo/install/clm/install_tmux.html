Gentoo GNU/Linuxの自作機(CLM)へのインストール: tmuxの導入

(工事中 0%)

インストール

# emerge -pv tmux

39パッケージがインストールされるとのこと。 Xの導入、設定はあとでするので今回はUSE="-X"としてみる。

# USE="-X" emerge -pv tmux

vimもあとで入れるのでUSE変数vim-syntaxも無効にする。

# USE="-X -vim-syntax" emerge -pv tmux

良さそうなのでこれでインストールする。

# USE="-X -vim-syntax" emerge -av tmux
