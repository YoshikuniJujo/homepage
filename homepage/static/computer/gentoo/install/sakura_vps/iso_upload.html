Gentoo Linux: さくらVPSへのインストール: SFTPでISOイメージをアップロード

(工事中 0%)

はじめに

SFTPでGentooのインストールISOイメージをアップロードする。

sftpコマンド

ISOイメージを保存したディレクトリに移動しておく。

% sftp -P 22 vpsAAAABBBBCCCC@vps-iso1.sakura.ad.jp
...

sftp> ls
dev iso
sftp> cd iso
sftp> put install-amd64-minimal-201XXXXX.iso
Uploading ...
install-amd64-minimal-201XXXXX.iso 100% 211MB ...
sftp>

まとめ

インストールISOイメージをアップロードした。
