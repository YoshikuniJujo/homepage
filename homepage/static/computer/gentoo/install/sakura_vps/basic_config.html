tmuxの導入

コンソールを分割できるとインストール作業がしやすい。

# emerge -pv tmux
# emerge -av tmux

今のところはvim-syntaxはオフでインストールする。 あとでemerge -avu --deep --newuse worldあたりでvim-syntaxがonで 再インストールされるだろう。

vim

# emrege -pv vim
# emrege -av vim

cpuinfo2cpuflags

# emerge -pv cpuinfo2cpuflags
# emerge -av cpuinfo2cpuflags
# cpuinfo2cpuflags-x86 >> /etc/portage/make.conf
