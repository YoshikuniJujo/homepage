Gentoo Linux: さくらVPSへのインストール: ISOイメージからブート

(工事中 0%)

はじめに

アップロードしたISOイメージからブートする。 ブート直後はさくらのコンソールから操作する。 SSHを設定したあとはsshで操作できるようになる。

ブート

ISOアップロード用アカウントを設定したページの右下の「設定内容を確認する」を クリックする。 問題がなければ「インストールを実行する」をクリックする。

コンソール

「VNCコンソール(HTML5版)を起動」をクリックする。

僕の場合vimperatorを使っているのでキーを無効にする必要がある。 Shift-Escだ。 また、GentooのインストールCDは時間がたつとハードディスクからブートしてしまうので いそいでReturnキーを押す必要がある。 僕が英語キーボードを使っているせいなのだろうが、 :の入力ができないのでvimの保存・終了にはZZを使うことになる。 sshを設定するまでの辛抱だ。

まとめ

とりあえずVPSコンソール上での作業はできる。 vimperatorが使いにくくなることと記号の入力に問題があるので とにかくSSHを設定することに全力を注ぐ。 sshで接続できれば気持ち良く作業ができる。
