Gentoo Linux: さくらVPSへのインストール: ISOイメージのダウンロード

(工事中 0%)

はじめに

Gentoo GNU/Linuxのインストールにはインストール作業を行う環境に入る必要がある。 そのためのブート可能なCDイメージをダウンロードする。

ダウンロード

Gentooのミラーサイトに行きISOイメージをダウンロードする。 日本だとjaist.ac.jpとiij.ad.jpとがある。 AMD64用のISOイメージとする。

  1. releases/
  2. amd64/
  3. autobuilds/
  4. current-install-amd64-minimal/
  5. install-amd64-minimal-201XXXXX.iso

これを書いている時点ではinstall-amd64-minimal-20150924.isoが最新だ。 ファイルのが正しいものであるかどうかを確認するために install-amd64-minimal-20150924.iso.DIGESTS.ascもダウンロードしておく。

ファイルの完全性の確認

% gpg --recv-keys 0xBB572E0E2D182910
% gpg --recv-keys 0xBB572E0E2D182910
% gpg --verify install-amd64-minimal-201XXXXX.iso.DIGESTS.asc
...
主鍵のフィンガー・プリント: 13EB BDBE DE7A 1277 5DFD B1BA BB57 2E0E 2910

フィンガー・プリントをGentoo署名ページで確認する。

% grep -A 1 -i sha512 install-amd64-minimal-201XXXXX.iso.DIGESTS.asc | grep '.iso$'
075212e58ddec6ef04791fc90c573c45 0b24b34ae38eccaf29cc6c17b7436148 e4ef4f133cf48d84aa597e02117b3bda 7086d2a7176224c208f71dba2622a913 install-amd64-minimal-20150924.iso
% sha512sum install-amd64-minimal-201XXXXX.iso
