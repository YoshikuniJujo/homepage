はじめに

SSHでのインストールを行う。 SSHでの常識として直接rootユーザでの接続は許可しない。 一般ユーザでログインしてsuでrootユーザとなる。 こうするとパスワードの認証が二重となりすこしセキュリテティが向上する。 一般ユーザを作成する。

コマンドuseradd

# useradd -m -G users,wheel yoshio
# passwd yoshio
...

キーボードの設定がうまくいっていないので今のところは長めの数字とアルファベットのみ のパスワードとする。
