はじめに

sshで接続したい。 インターネットに接続する必要がある。

インターネット設定のチェック

インターネットの設定をチェックする。 「サーバ設定」のページの「ネットワーク情報: IPv4」を参照する。

  • アドレス: 153.XXX.YYY.ZZZ
  • ゲートウェイ: 153.XXX.YYY.1
  • ネットマスク: 255.255.254.0
  • プライマリDNS: 210.188.224.10
  • プライマリDNS: 210.188.224.11

以下のようなコマンドを実行する。 このときbroadcastアドレスがアドレスと0.0.1.255との値理和になっていることに注意する。 これはネットマスクが255.255.254.0となっているためだ。

% ifconfig eth0 153.XXX.YYY.ZZZ broadcast 153.XXX.YYW.255 netmask 255.255.254.0
% route add default gw 153.XXX.YYY.1
% vi /etc/resolv.conf
nameserver 210.188.224.10
nameserver 210.188.224.11

vimは、僕の環境では:が入力できないのでZZで保存・終了とした。
