はじめに

ローカルなPCへのインストールと基本は同じだ。 ただし準備の段階で以下が必要になる。

  1. ISOイメージのダウンロード (工事中 0%)
  2. ISOイメージアップロード用のアカウントの作成 (工事中 0%)
  3. SFTPでISOイメージを/isoディレクトリ下にアップロード (工事中 0%)
  4. ISOイメージからブート (工事中 0%)
  5. インターネットへの接続 (工事中 0%)
  6. 一般ユーザ一の作成 (工事中 0%)
  7. SSHDの設定 (工事中 0%)
    • とくにポートをハイポートにしておく
    • rootユーザのパスワードの設定
  8. セキュリティを高めるためにRSA認証とする (工事中 0%)
    • RSA公開鍵認証の有効化とパスワード認証の無効化

またカーネルの設定時にvirtIOが使えるようにする必要がある。
