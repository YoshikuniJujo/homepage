はじめに

オープンな電子書籍フォーマットだ。 この出現によって「出版」という概念が大きく変わっていくように思う。 まずは電子書籍として世に出てそれを印刷所がオンデマンドで印刷するといった形だ。 電子媒体として読む場合には無料だが紙の書籍として読む場合には いくばくかの料金が発生する。 このモデルでは「読者を」ではなく「ファン」を増やす必要がある。 たとえば、「鑑賞用」と「保存用」と「布教用」に3枚のレコードを買うといった レベルのファンを増やすことだ。 1度読んで捨てるのではなく何度も何度も読みたくなるような書籍を作ろう。

  • 最低限の要素だけを使ったHello, EPUB!を作成する
