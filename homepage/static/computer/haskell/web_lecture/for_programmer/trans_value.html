単純な値の変換

(工事中 0%)

はじめに

単純な値を情報としての意味を変えずに関数にすることができる。

関数eight

eight.hs

「整数をとり何らかの値を返す関数」をとり その関数に引数として整数8を与えて何らかの値を返す関数を考える。

eight :: (Int -> a) -> a
eight f = f 8

この関数は以下のようにも書ける。

eight = ($ 8)

同じことだ。

関数eightは整数8とほとんど同じものだ。 対話環境で見てみよう。

% ghci eight.hs
*Main> eight (+ 5)
13
*Main> eight even
True
*Main> eight id
8

単純な値から関数への変換

value_fun.hs

どのような値であれ関数eightのような関数に変換することができる。

x :: X
funX :: (X -> b) -> b
funX = ($ x)

この変換を行う関数を書いてみよう。

valToFun :: a -> ((a -> b) -> b)
valToFun x = ($ x)

試してみる

*Main> :load value_fun.hs
*Main> let three = valToFun 3
*Main> three (+ 7)
10
*Main> three even
False
*Main> :m + Data.Char
*Main Data.Char> let c = valToFun 'c'
*Main Data.Char> c isLower
True

逆に単純な値と同じ意味を持つ関数を単純な値に変換する関数も書ける。

けど、ランク2多相拡張とかが必要になり、説明するのつらいので、ごまかす。 以下はそのうち消す。

funToVal :: ((a -> b) -> b) -> a

こっちを残す

関数から単純な値への変換は引数に関数idを与えればいい。
