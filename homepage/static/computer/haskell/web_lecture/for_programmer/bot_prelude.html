入力は書きこみの内容のみ

入力された文字列のみから返答を生成する。 入出力としてinteract :: (String -> String) -> IO ()のみを使うので、 それ以外の入力を受けとることができない。 もっと面白いことはIOをきちんと学んだらできる。

  • 真偽値
  • コンス
  • 変数定義
  • 関数定義
