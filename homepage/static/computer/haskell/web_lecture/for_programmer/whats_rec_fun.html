再帰関数とは

再帰という強力な概念

何かを表すときにそれ自身を参照することを再帰と呼ぶ。 再帰は関数型言語の魂だ。 「くりかえし」は再帰だ。樹構造も再帰だ。 再帰の理解は必須だ。 しかし、生の再帰は強力すぎてときにコードを「わかりにく」くする。 道具箱を見てより力の弱い概念を探す。 使えるものがなかったときに生の再帰を使う。 たとえば、単純な「くりかえし」には「列挙-蓄積」の枠組みが使える。

再帰の例

「自然数とは1または自然数に1を加えた数だ」。

再帰関数

定義に自身への参照を含む関数を再帰関数と呼ぶ。 Haskellでは自分で定義する関数も他の誰かが定義した関数も多くが再帰関数だ。
