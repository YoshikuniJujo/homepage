はじめに

deriving Showとすると自分で定義した型の値を「いい感じに」表示してくれる 関数showを使うことができるようになる。 derivingで指定できる値はShowだけでなくいくつかある。 今回はReadを学ぶ。

複数のderiving

derivingで複数の値を指定する場合は括弧でくくりコンマでくぎる。 ShowとReadを指定する場合には以下のようになる。

data Foo = Bar | Baz deriving (Show, Read)

Read

deriving Readとすると自分で定義した型の値を「いい感じに」読みこんでくれる 関数readが使えるようになる。 関数showによって得られた文字列を関数readによって読みこむともとの値が得られる。 ただし関数readの返り値は多相的なので型注釈が必要になる。

data Foo = Bar Bool Char | Baz Int Double deriving (Show, Read)

% ghci deriving_read.hs
*Main> show $ Bar False 'c'
"Bar False 'c'"
*Main> read it :: Foo
Bar False 'c'
*Main> show $ Baz 8 5
"Baz 8 5.0"
*Main> read it :: Foo
Baz 8 5.0
