Maybe型からの値の取り出し

はじめに

多相関数の例としてMaybe型から値をとりだす関数

  • fromMaybe
  • maybe

を見ていこう。

関数fromMaybe

動作

Just値のときはなかみをNothingのときはデフォルトの値を返す。

% ghci
Prelude> :m Data.Maybe
Prelude Data.Maybe> fromMaybe 0 (Just 8)
8
Prelude Data.Maybe> fromMaybe 0 Nothing
0
Prelude Data.Maybe> fromMaybe ' ' (Just 'c')
'c'

定義

fromMaybe _ (Just x) = x
fromMaybe d _ = d

Maybe型の値がJust値のときはなかみの値xを返す。 そうでなければ第1引数の値dを返す。

型

fromMaybe :: a -> Maybe a -> a

デフォルト値、Maybe型のなかみ、返り値はすべて同じ型aだ。

関数maybe

動作

関数fromMaybeはデフォルト値を決めて値をとりだす。 関数maybeはデフォルト値を決めて値に関数を適用する。

% ghci
Prelude> :m Data.Char
Prelude Data.Char> maybe False isLower (Just 'c')
True
Prelude Data.Char> maybe False isLower Nothing
False
Prelude Data.Char> maybe 0 ord (Just 'c')
99
Prelude Data.Char> maybe 0 ord Nothing
0

定義

maybe _ f (Just x) = f x
maybe d _ _ = d

Maybe型の値がJust値ならばなかみの値xに引数で与えられた関数fを適用する。 そうでなければデフォルト値を返す。

型

maybe :: b -> (a -> b) -> Maybe a -> b

第2引数はMaybe型のなかみの型aの値を型bの値に変換する関数だ。 デフォルト値の型は変換後の値の型と同じ型bとなる。 返り値も同じく型bだ。

まとめ

Maybe型から値を取り出す関数を見た。 関数fromMaybeと関数maybeはデフォルト値を指定することでNothing値に対応する。 fromMaybeはそのままの値をとりだす。maybeは関数適用によって値をとりだす。

fromMaybeはなかの値をいじらないことで多相性を維持している。 maybeでは値の操作を第2引数の関数にまかせることで多相性を維持している。 多相関数の多くは引数の関数で値の変換を行う。 つまり多相関数の多くは高階関数でもある。
