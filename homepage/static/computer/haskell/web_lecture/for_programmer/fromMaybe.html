Maybe型からの値の取り出し

はじめに

多相関数の例としてMaybe型から値を取り出す関数を見ていこう。 まずは関数fromMaybeを学び、次に関数maybeを学ぶ。

関数fromMaybe

Maybe型から値を取り出す関数fromMaybeがある。 デフォルトの値を決めておきMaybe型の値がNothingだった場合にはそれを返す。

% ghci
Prelude> :m Data.Maybe
Prelude Data.Maybe> fromMaybe 0 (Just 8)
8
Prelude Data.Maybe> fromMaybe 0 Nothing
0
Prelude Data.Maybe> fromMaybe ' ' (Just 'c')
'c'

定義

fromMaybeの定義は以下のようになる。

fromMaybe _ (Just x) = x
fromMaybe d _ = d

Maybe型の値がJust値だった場合は中身の値を返し、 そうでなかった場合は第1引数で指定したデフォルト値を返す。

型

fromMaybeの型は以下のようになる。

fromMaybe :: a -> Maybe a -> a

Maybe型の中身の型とデフォルト値の型、そして返り値はすべて同じである必要がある。 よってそれらを同じ型変数aとする。

関数maybe

関数fromMaybeはデフォルト値を決めて値を取り出す関数だ。 それに対して関数maybeはデフォルト値を決めて値に関数を適用する関数である。

% ghci
Prelude> :m Data.Char
Prelude Data.Char> maybe False isLower (Just 'c')
True
Prelude Data.Char> maybe False isLower Nothing
False
Prelude Data.Char> maybe 0 ord (Just 'c')
99
Prelude Data.Char> maybe 0 ord Nothing
0

定義

定義は以下のようになる。

maybe _ f (Just x) = f x
maybe d _ _ = d

Maybe型の値がJust値の場合には中身の値に引数で与えられた関数を適用し、 そうでなければデフォルト値を返す。

型

型は以下のようになる。

maybe :: b -> (a -> b) -> Maybe a -> b

Maybe型の中身の型aの値を型bの値に変換する関数を第2引数に取る。 デフォルト値は変換後の値と同じ型である必要があるので型はbとなり、 返り値も同様にb型の値となる。

まとめ

Maybe型から値を取り出す関数を見た。 関数fromMaybeと関数maybeは ともにデフォルト値を指定することでNothing値に対応している。 違いは前者は直接値を取り出し後者は関数適用後の値を取り出すという点だ。

前者は中身の値をいじらないことで多相性を維持しているのに対し、 後者は値の操作を引数として取った関数に任せることで多相性を維持している。 このように、多相関数の多くは引数として取った関数で値の変換を行う場合が多い。 よって多相関数はその多くが高階関数となる。
