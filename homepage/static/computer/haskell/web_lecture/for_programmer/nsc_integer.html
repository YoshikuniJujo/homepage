はじめに

整数値の入力と表示を実装する。 構文解析や値の評価をする関数も作成する。 整数値はそのままの整数値に評価されるので評価関数はスタブ的なものとなる。

型Value

Environment.hs

環境やそれに関する演算を置くためのモジュールEnvironmentに型Valueを定義する。 まずは整数値のみとする。

module Environment (Value(..)) where

data Value =
Int Integer

表示用の関数も用意しておこう。

showValue :: Value -> String
showValue (Int i) = show i

% ghci Environment.hs
*Environment> showValue $ Int 8
"8"

モジュールParse

Parse.hs

モジュールParseは関数parseを公開する。 またモジュールEnvironのValue型の値を使用する。 モジュールData.Charも導入する。

module Parse (parse) where

import Data.Char

import Environment

関数scheme

nsc.hs

Nano Scheme(以下nscとする)は複数の式を評価しその結果を1行ずつ出力することにする。 ソースコードの文字列と環境をとって複数の値と新しい環境を返す関数schemeを考える。 構文解析や評価中にエラーが生じる可能性を考えて返り値はMaybe型の値とする。

scheme :: String -> Env -> Maybe ([Value], Env)
