GHCi: 対話環境に触れる

はじめに

Haskell Platformをインストールする。 Windowsではここ、 Macではここに インストーラがある。 Linuxではディストリビューションごとにそれぞれのやりかたがある。 ここに 情報がまとまっている。 以下、Haskell Platformがインストールされているものとして話を進める。

ghcのチェック

ghcがインストールされているかどうかのチェック。 コマンドプロンプトを立ち上げて以下を入力。 先頭の'%'はコマンドプロンプトに対する入力であることを示すため。 これは入力しない。

% ghc --version
The Glorious Glasgow Haskell Compilation System, version X.Y.Z

ghcの名前とバージョンが表示されればghcはインストールされている。

終わらせかた

さっそく対話環境を立ち上げてみよう。 

% ghci
Prelude>

Prelude>はプロンプトである。 何事もまずは終わらせかたを知っておくべきだ。 そうでないと対話環境から抜けられなくて困惑するはめになる。 プロンプトに':q'を入力し続けてリターンキーを押す。 ':q'は':quit'の略。 省略せずに':quit'と入力しても良い。 

Prelude> :q

値の表示

再度、ghciを立ち上げていろいろな値を入力してみよう。 

% ghci
Prelude> 4492
4492
Prelude> 'c'
'c'
Prelude> pi
3.141592653589793

対話環境は入力した式を評価し結果を表示する。 4492や'c'はリテラルでありそのまま出力される。 piは定義済みの変数だ。円周率の値が表示される。

定義ファイルの読み込み

学習の流れ

入出力について学ぶまでは以下のような形で学習を進める。

  1. 関数を定義したファイルを作成
  2. GHCiに読み込む
  3. 関数を評価し結果を表示
  4. ファイルを編集
  5. 変更したファイルの再読み込み
  6. 関数を評価し結果を表示
  7. 4から6をくりかえす

リンゴが好き

ファイルの読み込み、再読み込みの練習。 fruits.hsを作成する。

fruits.hs

myFavoriteFruit = "apple"

引数としてファイル名を指定してghciを立ち上げる。

ghci fuits.hs
*Main>

myFavoriteFruitと入力しリターンキーを押す。

*Main> myFavoriteFruit
"apple"

バナナが好き

リンゴよりもバナナのほうが好きかもしれない。 対話環境はそのまま走らせておく。 エディタでfruit.hsを編集。

fruits.hs

myFavoriteFruit = "banana"

試してみよう。

*Main> myFavoriteFruit
"apple"

「おや?変更が反映されてない!」。 ファイルを変更した後には再読み込みだ。 ':r'または':reload'と入力。 

*Main> :reload
*Main> myFavoriteFruit
"banana"

まとめ

Haskell Platformをインストールすれば、コンパイラ、インタプリタ、対話環境だけでなく、 パッケージ管理ソフトやドキュメントの自動生成ソフト、標準的なライブラリがそろう。 入出力について学ぶまでは対話環境で学習を進める。 対話環境はghciコマンドで開始、':q'で終了。 定義ファイルを読み込むにはそのファイル名を引数にしてghciを立ち上げる。 ファイルを編集したら:reloadで再読み込みする。

課題

  1. ghciを立ち上げて自分の生まれた日を入力してみよう
  2. 続いて自分の名前のイニシャルを入力する
    • 文字は''で囲んで入力する
  3. myFavoriteColor変数を定義したファイルを作成する
    • まずは2番目に好きな色を指定してみよう
    • 対話環境に読み込み表示する
    • 1番好きな色に変更する
    • 再読み込みして再度表示してみよう
