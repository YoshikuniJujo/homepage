鍵を固定した関数

同じ鍵でいろいろなメッセージを暗号化、復号することを考える。 そのようなときにいちいち鍵を引数として与えるのは面倒だ。 鍵を固定した関数を作成しよう。 rsa.hsに以下の関数を追加する。 

encrypt1 m = encrypt 138689 13 m
decrypt1 c = decrypt 138689 95497 c

試してみる。

*Main> :reload
*Main> encrypt1 4492
47255
*Main> decrypt1 47255
4492
