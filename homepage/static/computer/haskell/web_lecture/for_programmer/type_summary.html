まとめ: 型

型とは何か

型はその値で何ができるかを示す。 具体的には以下の2つが型によって決まる。

  • どの関数が適用可能か
  • どの関数がその値を返すか

例

Bool

Char

Integer

試してみる

Prelude> :t True
True :: Bool
Prelude> :t 'c'
'c' : Char

関数の型

関数も値なので型を持つ。 関数の型は引数の型と返り値の型から成る。

書式

[引数の型] -> [返り値の型]

多引数関数

多引数関数に見えるものは関数を返す関数である。

[引数1の型] -> ([引数2の型] -> [返り値の型])

'->'が右結合なので丸括弧が省略できる。

[引数1の型] -> [引数2の型] -> [返り値の型]

例

Char -> Bool

Char -> Integer -> Bool

試してみる

Prelude> :m Data.Char
Data.Char> :t isDigit
Char -> Bool

型宣言

書式

[変数], [変数], ... :: [型]

例

num1, num2 :: Integer

fun :: Char -> Integer -> Bool

型注釈

多相的な値に型注釈をつけることで型を定めることができる。

書式

[値] :: [型]

例

8 :: Integer

8 :: Double

試してみる

Data.Char> 8 :: Integer
8
Data.Char> :t it
it :: Integer
Data.Char> 8 :: Double
8.0
Data.Char> :t it
it :: Double

モジュールの読み込み

書式

import [モジュール名]

import [モジュール名] ([インポートリスト])

import qualified [モジュール名] as [別名]

使い捨てのコード以外では2個目、3個目の形を使ったほうが良い。
