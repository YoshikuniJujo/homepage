型とは何か

型とは「何ができるか」である

値には型がある。型は以下の2つのことを示す。

  • 値にどの関数が適用できるか
  • どの関数がその値を返すか

型は値で何ができるかを示す。

Char型の値は小文字にできる

文字を表す値はChar型である。 Char型の値はtoLowerという関数で小文字にできる。 

% ghci
Prelude> :m Data.Char
Prelude Data.Char> :t 'L'
'L' :: Char

toLowerはData.Charモジュールが公開している。 対話環境でのモジュールの読み込みは':m'コマンドだ。 値の型を表示するには':t'コマンドを使う。 ':module', ':type'の略記だ。 文字'L'を表す値の型はChar型だ。 

Prelude Data.Char> toLower 'L'
'l'

Char型の値はtoLower関数で小文字にできる。 正確にはtoLower関数で小文字の値と関連づけられるということ。 値'L'自体は変化しない。

Bool型の値は小文字にできない

FalseやTrueという値の型はBool型でありChar型ではない。 小文字にすることはできない。

Prelude Data.Char> :t False
False :: Bool
Prelude Data.Char> toLower False

<interactive>:X:Y:

「型が一致してないよ」と怒られる。
