型とは何か

型とは「何ができるか」である

値には型がある。 型は以下の2つのことを示す。

  • その値にどの関数が適用できるか
  • どの関数がその値を返すか

その値を使って何ができるかを示すのが「型」である。

Char型は小文字にできる

文字を表すChar型という型がある。 Char型の値はtoLowerという関数を使って小文字にすることができる。 toLowerを使うにはData.Charモジュールを読み込む必要がある。 対話環境でモジュールを読み込むには':m'コマンドを使う。 また':t'コマンドで値の型を表示することもできる。 それぞれ':module'と':type'の略記だ。 

% ghci
Prelude> :m Data.Char
Prelude Data.Char> :t 'L'
'L' :: Char

文字'L'という値の型はChar型であることがわかる。

Prelude Data.Char> toLower 'L'
'l'

Char型の値はtoLower関数で小文字にすることができる。 正確に言うと、toLower関数で小文字の値と関連づけられているということであり、 'L'自体が変化しているわけではない。

Bool型は小文字にできない

TrueやFalseという値はBool型の値である。 これはChar型の値ではないので小文字にすることはできない。

Prelude Data.Char> :t False
False :: Bool
Prelude Data.Char> toLower False
