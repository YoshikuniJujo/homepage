そのままにする関数

関数id

動作

% ghci
Prelude> id 8
8
Prelude> id "hello"
"hello"
Prelude> id 'c'
'c'

関数idは引数と同じ値を返す。 id 8は8である。 値の型を問わずすべての値で「同じ値を返す」ことができる。 つまり関数idはすべての型の値が引数となる。

定義

id x = x

使い道

ここまでの説明で理解できる例で使い道を示す。 第1引数で第2引数を除算した値を返す関数を作る。 第1引数が0のときは第2引数をそのまま返すことにする。

devide :: Double -> Double -> Double
devide 0 b = b
devide a b = b / a

2行目は演算子の部分適用でdevide a = (/ a)とできる。 「devide aは引数をaで割る関数」だ。 1行目はどうなるだろう。 ここで関数idが使える。devide 0 = idだ。 「devide 0は引数をそのまま返す関数」だ。

id.hs

devide :: Double -> Double -> Double
devide 0 = id
devide a = (/ a)

仮引数を減らすことでより「直接的」な表現にできる。

% ghci id.hs
*Main> devide 6 9
1.5
*Main> devide 0 8
8.0

型

関数idの型はどうなるだろう。 Integer型の値8に対して適用された(id 8)とき関数idの型は以下のようになる。 

id :: Integer -> Integer

文字列"hello"に適用された(id "hello")ときは以下のようになる。

id :: String -> String

[型X] -> [型X]のような形だ。 引数と返り値の型が同じであれば良い。 「型変数」を使う。 型変数は小文字で始まる識別子でありしばしば小文字1文字で表される。

id :: a -> a

型aの値を取って同じ型aの値を返す。確認してみる。

% ghci
Prelude> :t id
id :: a -> a

まとめ

関数idはもっとも単純な多相関数と言える。 引数をそのまま返す関数だ。 使い道がないように思えるが有用な関数だ。 いろいろな用途に使える。
