関数の型

関数も値

Haskellでは関数は「値」である。 より具体的に言えば関数は「第一級オブジェクト」である。 だから、関数も「型」を持つ。

関数の型の表現

関数の型は引数の型と返り値の型から成る。 以下のように表現される。 

[引数の型] -> [返り値の型]

Haskellには1引数関数しかないという話を覚えているだろうか。 2引数関数に見えるものは本当は「関数を返す関数」である。 よって以下のようになる。 

[引数1の型] -> ([引数2の型] -> [返り値の型])

'->'は右結合なので丸括弧は省略できる。

[引数1の型] -> [引数2の型] -> [返り値の型]

丸括弧を省略すると2引数関数のように見えてくる。

関数の型の例

関数toLower

関数toLowerは引数として文字を取り、小文字の値を返す関数。 引数、返り値ともにChar型なのでその型は以下のようになる。 

Char -> Char

関数xxcrypt

関数xxcryptは整数値を3つ取って、整数値を返す関数である。 代表的な整数値のひとつにIntegerがある。 よって以下のようになる。 

Integer -> (Integer -> (Integer -> Integer))

整数を取って関数を返す型となっている。 '->'が右結合であることを利用して丸括弧を省略すると3引数関数のように見える。 

Integer -> Integer -> Integer -> Integer
