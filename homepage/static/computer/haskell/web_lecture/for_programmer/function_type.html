関数の型

関数は値

関数は「値」だ。 「第一級オブジェクト」だ。 関数にも「型」がある。

関数の型とは

関数の型は引数の型と返り値の型から成る。 '->'で引数の型と返り値の型をつなぐ。 

[引数の型] -> [返り値の型]

Haskellには1引数関数しかない。 2引数関数に見えるものは「関数を返す関数」だ。 

[引数1の型] -> ([引数2の型] -> [返り値の型])

'->'は右結合なので丸括弧は省略できる。

[引数1の型] -> [引数2の型] -> [返り値の型]

丸括弧を省略すると2引数関数のようになる。

例

関数toLower

関数toLowerは引数として文字を取り小文字の値を返す。 引数、返り値ともにChar型である。 

toLower :: Char -> Char

確認してみよう。

% ghci
Prelude> :m Data.Char
Prelude Data.Char> :t toLower
toLower :: Char -> Char

関数xxcrypt

関数xxcryptは整数値を3つ取って整数値を返す。 整数型としてここではIntegerを考えよう。 

Integer -> (Integer -> (Integer -> Integer))

整数をとって「整数をとって「整数をとって整数を返す関数」を返す関数」を返す関数だ。 '->'は右結合なので丸括弧を省略すると3引数関数のようになる。 

Integer -> Integer -> Integer -> Integer
