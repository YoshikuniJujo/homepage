引数を入れ替える

関数flip

動作

関数の第1引数と第2引数を入れ換えたいときがある。 第2引数に部分適用するときなどだ。 関数flipが定義されている。

% ghci
Prelude> mod 8 3
2
Prelude> flip mod 3 8
2
Prelude> let mod3 = flip mod 3
Prelude> mod3 8
2

定義

flip :: (a -> b -> c) -> b -> a -> c
flip f x y = f y x

第1引数に2引数関数fを取る。 第2、第3引数の値を逆にして関数fを適用する。

演算子の部分適用

演算子は(`mod` 3)のように第2引数に部分適用できる。 同じことが関数flipでできる。

flip mod 3

(/ 3)を以下のように書ける。

flip (/) 3

まとめ

関数flipは2引数関数の2つの引数を入れ替える。 この関数は演算子の部分適用のときに暗黙に使われているとも考えられる。
