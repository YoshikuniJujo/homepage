型宣言

記法

関数や値の定義の際に型宣言をつけられる。

[変数名] :: [型]

略記法

同じ型の変数をいくつか定義するための略記法がある。

[変数名1], [変数名2], [変数名3] :: [型]

演算子

演算子は丸括弧で関数に変換する。

例

num1 :: Integer
str1 :: String
chr1, chr2, chr3 :: Char
xxcrypt :: Integer -> Integer -> Integer -> Integer
(<!>) :: String -> Integer -> Bool

型宣言をつける意義

型宣言は任意だ。 型宣言がなければコンパイラが型を推論してくれる。 しかし、以下のような理由から、 完全に使い捨てのスクリプト等でなければ型宣言はつけるべきだ。

  • 何を作っているのかが明確になる
  • バグの多くは型の不一致をともなう
  • 型宣言はドキュメントになる
  • 複雑な型だと型推論が利かないことがある
