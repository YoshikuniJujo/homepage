型宣言

記法

[変数名] :: [型]

略記法

同じ型の変数を複数定義する略記法がある。

[変数名1], [変数名2], [変数名3] :: [型]

演算子

演算子は丸括弧で関数に変換する。

例

num1 :: Integer
str1 :: String
chr1, chr2, chr3 :: Char
xxcrypt :: Integer -> Integer -> Integer -> Integer
(<!>) :: String -> Integer -> Bool

型宣言の意義

型宣言は任意だ。型宣言がなければコンパイラが型を推論してくれる。 しかし、使い捨てのコードでなければ型宣言はつける。

  • 何を作っているのかが明確になる
  • バグの多くは型の不一致をともなう
  • 型宣言はドキュメントになる
  • 複雑な型だと型推論が利かないことがある
