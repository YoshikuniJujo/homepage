型宣言

記法

Haskellでは関数や値を定義する際に型宣言をつけることができる。 記法は以下のようになる。 

[変数名] :: [型]

略記法

同じ型の変数をいくつか定義する場合には以下のように書くことができる。

[変数名1], [変数名2], [変数名3] :: [型]

演算子

演算子は丸括弧で囲ってやることで関数名として、型宣言をする。

例

num1 :: Integer
str1 :: String
chr1, chr2, chr3 :: Char
xxcrypt :: Integer -> Integer -> Integer -> Integer
(<!>) :: String -> Integer -> Bool

型宣言をつける意義

Haskellでは型宣言はオプショナルである。 型宣言がない場合、コンパイラが型を推論してくれる。 しかし、以下のような理由から、 完全に使い捨てのスクリプト等でないならば、型宣言はつけるべきだ。

  • 自分が何を作っているのかが明確になる
  • バグの多くは型の不一致をともなう
  • 型宣言はドキュメントとしても優れる
  • 拡張機能を使う場合、型推論が利かない場合がある
