型注釈

型注釈とは

'式 :: 型'という形で値の型を指定することができる。 多相型を学ぶとわかるが、 たとえば、対話環境で表示するときに型が決まらないことがある。 そんなときは式に型注釈をつければ良い。

数の例

多相型や型クラスを学んでからきちんと説明するが、 実はリテラルの「数」は型が不定である。 対話環境などで、必要なときは型注釈をつけて型を定めてやると良い。

% ghci
Prelude> 8 :: Integer
8
Prelude> :t it
it :: Integer
Prelude> 8 :: Double
8.0
Prelude> :t it
it :: Double
