型注釈

型注釈とは

'式 :: 型'という形で値の型を指定することができる。 多相型を学ぶと理解できると思うが、 たとえば、対話環境で表示する際に型が決まらないことがある。 そのようなときに式に対して':: 型'という形で型注釈をつけてやる必要がある。

数の例

今はまだちゃんと理解できなくても良いが、 実はリテラルの「数」はそのままでは型が定まっていない。 よって、型注釈をつけてやることで、様々な型の値として扱うことができる。

% ghci
Prelude> 8 :: Integer
8
Prelude> :t it
it :: Integer
Prelude> 8 :: Double
8.0
Prelude> :t it
it :: Double
