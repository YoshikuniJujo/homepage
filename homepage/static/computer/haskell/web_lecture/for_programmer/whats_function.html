関数とは何か

プログラマのかたが対象なので「関数とは何であるか」よりも「関数とは何ではないか」を 強調する必要がある。

関数とは値を他の値に関連づけるものである。 中学か高校くらいのときにy = f(x)という形で学んだと思うが、 それが関数である。

関数とは手続きをまとめたものではない。 「ユーザに入力をうながし、入力値を取得し、それを演算し、出力に書き出す、 そのような動作をまとめたもの」は関数ではない。 複数の処理をまとめたものが関数なのではない。

関数とは引数を与えると返り値を返すものである。 引数を返り値に対応づける際に何らかの演算や処理が行われるが、 どのような処理が行われるかは問題ではない。 関数というものにとって重要なのは引数と返り値であり、 そのあいだに行われる演算は無視することができる。
