関数concatMapによる関数filterの定義

はじめに

関数concatMapで関数filterが定義できる。

型

filterC :: (a -> Bool) -> [a] -> [a]

定義

filterC p = concatMap $ \x -> if p x then [x] else []

mapの段階とconcatの段階をわけて考えるとわかりやすい。 「引数xが条件pを満たすならば[x]にそうでないならば[]に」という変換を リストのすべての要素に対して行う。 たとえば条件が「偶数である」ならば以下のようになるだろう。

[3, 8, 9, 4, 2, 5]

が

[[], [8], [], [4], [2], []]

となる。これを関数concatでつなげてやる。

[8, 4, 2]
