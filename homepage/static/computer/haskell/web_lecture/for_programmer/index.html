はじめに

Haskellを学ぶのは海の向こうの国を知ることだ。 ただの関数型ではなく「純粋関数型」だ。

第一級関数、参照透過性、遅延性、静的型付けなどが特徴だ。 デザインの美しさと機能性の有機的な結合がある。
