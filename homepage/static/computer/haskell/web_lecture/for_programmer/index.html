はじめに

Haskellを学ぼう。 「Haskellを学ぶ」ということは「新たな言語を学ぶ」ということ以上の価値がある。 手続き型言語しか...

対象

プロ、アマチュアを問わず、何らかの言語でプログラムを組んだことのある人。 「変数」「引数」「返り値」「関数」という言葉の だいたいの意味がわかりHaskell入門を希望する人が対象。

必要な環境

ghcが必要。 ghcはHaskellのコンパイラ、インタプリタ、対話環境。 その他のツールやパッケージも含めてHaskell Platformとしてまとまっている。 これをインストールすること。 Windows, Mac, Linuxのそれぞれのソース、バイナリがある。
