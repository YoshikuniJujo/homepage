関数合成

演算子(.)

動作

演算子(.)は関数を合成する。 f . gは引数にgを適用した結果にfを適用する関数だ。

(* 2) . (+ 3)

% ghci
Prelude> ((* 2) . (+ 3)) 8
22

8に3を足して2をかけている。

定義

(.) f g x = f (g x)

第3引数xに第2引数である関数gを適用した結果に第1引数であるfを適用する。 演算子であることを強調したければ以下のようにも書ける。

(f . g) x = f (g x)

あるいは

f . g = \x -> f (g x)

使い道

RSA暗号の例の関数xxcryptを変形する。

xxcrypt n ed mc = mc ^ ed `mod` n

部分適用と関数合成で以下のように書き換えられる。

xxcrypt n ed = (`mod` n) . (^ ed)

「ed乗してnの剰余を取る」ということだ。

型

f . g

わかりやすさのために関数的な書きかたにする。

(.) f g

g、fの順に適用するのでgの返り値とfの引数は同じ型となる。 gの型をa -> bとしfの型をb -> cとする。 結果としてできるf . gはgの引数を引数としfの返り値を返り値とする。

f :: b -> c
g :: a -> b
f . g :: a -> c

演算子(.)はf, gを引数としf . gを返り値とする。

(.) :: (b -> c) -> (a -> b) -> (a -> c)

まとめ

関数合成演算子(.)を学んだ。 これを使うと「関数gを適用しそれに関数fを適用する」といった処理を直接的に書ける。
