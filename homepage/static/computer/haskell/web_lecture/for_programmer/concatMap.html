関数concatMap

動作

mapした結果をconcatする。

% ghci
Prelude> concatMap (replicate 3) "hello"
"hhheeellllllooo"

定義

concatMap' :: (a -> [b]) -> [a] -> [b]

関数concatと関数map fとを合成する。

concatMap' f = concat . map f

引数fは消せる。

concatMap' f = (concat .) $ map f

concatMap' f = (concat .) . map $ f

concatMap' = (concat .) . map

課題
