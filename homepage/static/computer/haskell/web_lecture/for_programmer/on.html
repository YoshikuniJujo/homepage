2つの引数に同じ関数を適用

はじめに

多相関数をうまく使うとコードをスマートにすることができる。 関数を組み合わせるやりかたそのものを、関数として定義することができるからだ。 たとえば「2つの値に同じ関数を適用してから2引数関数に渡す」という 関数の組み合わせかたが関数onとして定義されている。

関数on

動作

2つの値に同じ関数を適用してから2引数関数に渡す。 「2つの数のそれぞれを3で割った余りの比較」する例を示す。

on (==) (`mod` 3) 8 11

8を3で割った余りと11を3で割った余りとの比較だ。 対話環境で試してみよう。 

Prelude> :m Data.Function
Prelude Data.Function> on (==) (`mod` 3) 8 11
True

定義

定義は以下のようになる。

on :: (b -> b -> c) -> (a -> b) -> a -> a -> c
on op f x y = op (f x) (f y)

以下のようにも書ける。

on op f x y = f x `op` f y

まとめ

関数onは、たとえばソートのために大小比較をする際に比較される2つの値を、 事前に変換する際等にも使うことができる。

課題

  1. 3引数関数の3つの引数すべてに同じ関数を適用する関数on3を作成せよ
