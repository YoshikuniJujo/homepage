2つの引数に同じ関数を適用

はじめに

多相関数をうまく使うとコードをスマートにできる。 関数を組み合わせるやりかたそのものを関数として定義してそれを使う。 たとえば「2つの値に同じ関数を適用してから2引数関数に与える」という 関数の組み合わせかたが関数onとして定義されている。

関数on

動作

2つの値に同じ関数を適用してから2引数関数に与える。 「2つの数のそれぞれを3で割った余りの比較」をする例を示す。

on (==) (`mod` 3) 8 11

8を3で割った余りと11を3で割った余りとの比較だ。 対話環境で試してみよう。

Prelude> :m Data.Function
Prelude Data.Function> on (==) (`mod` 3) 8 11
True
Prelude Data.Function> ((==) `on` (`mod` 3)) 8 11
True

中置記法で書くと(`mod` 3)のうえで(==)を行うといった感じを強調できる。

定義

on op f x y = op (f x) (f y)

同じことを中置記法で定義すると以下のようになる。

(op `on` f) x y = f x `op` f y

型

on :: (b -> b -> c) -> (a -> b) -> a -> a -> c

a型の値の2つにそれぞれに対して(a -> b)型の関数を適用し、 得られたb型の値2つを(b -> b -> c)型の2引数関数にわたすと結果として c型の値が得られる。

使い道

関数onはソートのために大小比較をするときに比較される2つの値を 事前に変換するときに使える。

課題

  1. 3引数関数の3つの引数すべてに同じ関数を適用する関数on3を作成せよ
