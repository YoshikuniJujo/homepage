2つの引数に同じ関数を適用

(工事中 40%)

はじめに

多相関数をさらにいくつか学ぼう。

関数on

動作

2つの値に同じ関数を適用してから2引数関数に渡す。 「2つの数のそれぞれを3で割った余りの比較」などだ。

on (==) (`mod` 3) 8 11

8を3で割った余りと11を3で割った余りとの比較だ。 対話環境で試してみよう。 

Prelude> :m Data.Function
Prelude Data.Function> on (==) (`mod` 3) 8 11
True

定義

定義は以下のようになる。

on :: (b -> b -> c) -> (a -> b) -> a -> a -> c
on op f x y = op (f x) (f y)

以下のようにも書ける。

on op f x y = f x `op` f y
