変形

以下のような変形が可能であることを示す。

[ 結果 | ..., (x, y) <- [ (x, y) | x <- 式1, y <- 式2 ], ... ] => [ 結果 | ..., x <- 式1, y <- 式2, ... ]

左結合と右結合

concatMap

メモ

concatMap :: (a -> [b]) -> [a] -> [b]

cm = flip concatMap

[a] `cm` k = k a
m `cm` (: []) = m
m `cm` (\x -> k x `cm` h) = (m `cm` k) `cm` h
