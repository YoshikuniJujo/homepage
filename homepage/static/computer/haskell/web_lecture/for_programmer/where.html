構文: where

例

where構文は関数定義につくので対話環境からだと試しにくい。 ファイルに書き込んでから対話環境に読み込もう。

where.hs

v1 = x ^ x
where
x = 5

v2 = x * y
where
x = 3
y = 5

% ghci where.hs
*Main> v1
3125
*Main> v2
15

構文

以下のような対応となる。

foo = let var = val in [varを使った式]

foo = [varを使った式] where var = val

let ... in ...との違い

定義が後に来る

where節は関数定義の後ろにつけることができる。 これによって細かい定義を後回しにすることができる。

スコープ

where節は複数のガードをまたぐスコープを持つ。 これはlet ... in ...ではできない。

まとめ

let ... in ...とほぼ同じ働きをするwhere節について見た。 関数定義や変数定義にくっつけることができる。 スコープの広さの違い等があるが意味論的にはlet ... in ...と同じだ。 使いわけよう。
