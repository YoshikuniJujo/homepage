Hackage: 手動でのドキュメントのアップロード

問題

Hackageにパッケージをアップすると、しばらくして自動的にドキュメントが作成される。 しかし、最近、それがうまく動いていないことが多い。 そのような場合に手動でドキュメントをアップロードすることができる。 そのための方法を調べた。

maoeさんのブログで同じような現象を扱っている。

解決

ドキュメントのアーカイブを作成する

  1. cabal haddock --hyperlink-source
  2. cd dist/doc/
  3. tar -c -v -Hustar -f [アーカイブ名].tar [パッケージ名]

作成したアーカイブをアップロード

  1. 自分のパッケージのページに行く
  2. edit package informationをクリック
  3. Manage documentatin for以降の適切なバージョンをクリック
  4. 上で作成したtarアーカイブをアップロードする
