バイナリ解析器: MaybeからEither Stringに

Analyzer型の変更

MaybeではなくEither Stringを使うようにする。

data Analyzer a b =
	Analyzer { runAnalyzer :: a -> Either String (b, a) }

その他の関数の変更

ghciで走らせようとすると型エラーが報告されるので、 ひとつずつMaybeをEither Stringに、JustをRightに、N othingをLeft [メッセージ]に変更していく。
