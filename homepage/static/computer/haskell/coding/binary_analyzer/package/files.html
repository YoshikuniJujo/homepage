ライセンスについて

フリーソフトウェアファウンデーションによるライセンスの説明

LICENSEはライセンスを記述したファイル。 今、メジャーなのは三条項BSDライセンスかGPLライセンスあたりかと思う。 どっちがいいのかよくわからないが、 わかりやすいので僕自身は三条項BSDライセンスを主に利用している。 まちがえて四条項BSDライセンスにしてしまうと、使う人が困ってしまうので、 注意が必要だ。

BSD 3-Clause License BSD 2-Clause License

今、見たらフリーソフトウェア財団はGPL以外ではApache 2.0ライセンスを 推奨しているようだ。 これはサブマリン特許への対策を含むライセンスとのことだ。 すこし調べてみる必要がある。

Apache License, Version 2.0
