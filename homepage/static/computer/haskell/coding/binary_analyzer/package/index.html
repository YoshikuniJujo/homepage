バイナリ解析器: パッケージ化

(工事中 10%)

動機

AnalyzerモジュールはASN.1のBER方式のデコードに使用している。 なので、ここをブラックボックスとして扱えるとよりきれいに考えられる。 ブラックボックスとして考えるために、パッケージ化して、 Hackageにアップロードして、システムにインストールしてしまおうと思う。

価値

もちろんHackageを個人的なコードで汚すのは僕の本意ではない。 多少そのようなことをしてしまっている気もするが、本意ではない。 なのでAnalyzerというモジュールをHackageに加えることの価値について考える。 一番はやはり、バックトラックを行うパーサというものは、 バックトラックをそのまま実装するわけに行かず、 そこらへんの効率化によってコードが複雑化してしまっているということだろう。 バックトラックを必要としないバイナリのパースに使うにはオーバースペックだ。

よって、シンプルで無駄のない「バックトラックをしないパーサ」を 公開するのは意味あることかと考える。
