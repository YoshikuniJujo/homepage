バイナリ解析器: これは何か?

パーサのようなもの。 一般的なパーサは何らかのアルゴリズムでバックトラック的な動作を実現している。 つまり、後のほうのトークンが前のほうのトークンの解釈に影響を与える。 これは多くのテキスト形式のデータを解釈するのに必要だ。 しかし、バイナリデータの多くは前のトークンは後のほうのトークンの解釈に影響を与える が、その逆はないように設計されている。 これはテキスト形式のデータは人間が読み書きすることを考慮して、 人間の得意とする文脈判断を生かせるような設計になっているのに対して、 バイナリデータは機械が読むことを前提に、機械の苦手なバックトラックをなくすような 設計にしているためだろう。
