Lispっぽく表記: 関数定義の追加

(工事中 0%)

動機

変数束縛が出来、lambdaも使えるので、これで関数定義はできる。 以下のようにすれば良い。 

(define add (lambda (x y) (+ x y)))

しかし、やはり関数定義用の構文が使えたほうが「人間にとって」はわかりやすい。

構文

Lispでの関数定義の構文がどんな感じだったのか忘れたので調べてみた。 以下のような形らしい。

Common Lisp

(defun 関数名 (引数のリスト) 式)

Scheme

(define (関数名 引数のリスト) 式)

結論

Schemeの表記のほうが僕好みなので、そちらを採用する。
