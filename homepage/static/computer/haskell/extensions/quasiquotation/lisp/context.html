はじめに

型変数には文脈を指定することができる。 「文脈」とはその型変数がどの型クラスに所属するのかを示すものである。 たとえば以下のようになる。 

print :: Show a => a -> IO ()

記法としては以下のようにする。

(type print (=> ((Show a)) (-> a (IO ())))

字句解析

まずは字句解析器で'=>'をトークンとして読み込む必要がある。
