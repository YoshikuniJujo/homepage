Quasiquotation: Lispっぽく表記する

(工事中 2%)

何をする?

Quasiquotationの例としてHaskellのソースをLispっぽく表記する例を作成してみる。 基本的に「なんちゃって」なので、細かいところは作り込まないことにする。

この題材を選んだ理由

まず、表記だけLispっぽくするということで 意味論的な話はHaskellのものをそのまま使えるということ。 そして、ライブラリや自動生成を使わないでパーサを書く場合、 Lispの構文だと簡単であるということ、だ。

それでは始めよう

  • 字句解析器を作る (工事中 20%)
  • パーサを作る (工事中 0%)
  • QuasiQuoterを作る (工事中 0%)
  • 試してみる (工事中 0%)
