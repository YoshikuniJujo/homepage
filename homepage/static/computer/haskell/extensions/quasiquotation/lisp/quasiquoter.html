QuasiQuoterの定義

今のところ、使うのは式と宣言の部分のクォートなので、その2ヶ所だけ埋めておく。 

lisp :: QuasiQuoter
lisp = QuasiQuoter {
	quoteExp = fst . parseExp . lexer,
	quotePat = undefined,
	quoteType = undefined,
	quoteDec = parseDec . lexer }

試してみる

useLispLike.hs

[lisp|

(setq main (putStrLn (++ "Hello, " "world!")))

|]
% ghci useLispLike.hs
*Main> :set -XQuasiQuotes
*Main> main
Hello, world!
*Main> [lisp| (+ 3 (* 2 4)) |]
11
