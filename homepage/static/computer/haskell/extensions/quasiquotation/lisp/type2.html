Parser.hs

リストのパースのところで角括弧の字句解析は実装した。 ので、ここですることは単にOB, CBの並びを見つけてlistTに変換するだけだ。 

...
parseType (OB : CB : ts) = (listT, ts)
...

[Int]のような書きかたもパースできるようにすることにする。 

...
parseType (OB : ts) = let
	(tp, CB : ts') = parseType ts in
	(listT `appT` tp, ts')
...
