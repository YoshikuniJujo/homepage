WikipediaのHaskell 1.0の記述に 「実験的な機能の付加や統合を通じてHaskell 98の拡張や派生物を作成することを、 委員会ははっきりと歓迎した」とある。 実際にGHCには多くの拡張機能があり、 そのうちのいくつかはHaskellの新しい標準に取り込まれた。

WikipediaのHaskell 2010の説明に取り込まれた拡張機能が列挙されている。
