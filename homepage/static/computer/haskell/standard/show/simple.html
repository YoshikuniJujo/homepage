単純なShowクラスのインスタンスの例

simpleShow.hs

括弧の面倒は見てくれない

最も簡単なやりかたはクラス関数show定義してやることだ。

instance Show MyT where
	show (MyT i b c) = "MyT " ++
		show i ++ " " ++ show b ++ " " ++ show c
% ghci simpleShow.hs
Main> MyT 8 False 'c'
MyT 8 False 'c'
Main> Just it
Just MyT 8 False 'c'

この場合、括弧の面倒は見てくれない。
