パッケージ: sqlite-simplest

(工事中 0%)

はじめに

最低限の機能だけが使えるsqliteライブラリを作成する。

必要な機能

  • 接続と切断
  • テーブルの作成
  • 要素の追加
  • 要素の更新
  • キーとスロットを指定して値を取得

型

  • newtype SQLite3
  • data Table = Table ByteString
  • data Keys = { primaryKey :: ByteString, keys :: [ByteString] }

関数

  • open :: IO SQLite3
  • close :: SQLite3 -> IO ()
  • table :: SQLite3 -> Table -> Keys -> IO ()
  • insert :: SQLite3 -> Table
