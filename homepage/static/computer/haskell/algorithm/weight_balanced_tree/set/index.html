リストでの集合の実装

集合の主な機能は要素の追加、要素の削除、要素が集合内に存在することの確認である。 また要素の追加の初期値として空集合が必要だ。 リストで集合を表現した場合、空集合、追加、削除、存在確認は それぞれ[]、(:)、filter . (/=)、elemとなる。 この場合実行効率はそれぞれO(1)、O(1)、O(n)、O(n)だ。 要素の削除や存在確認にかかる時間は集合に含まれる要素の数が多くなると それに比例して大きくなってしまう。
