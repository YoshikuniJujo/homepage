Theorems for free! (PDF)

概要

多相関数の型からその関数の満たす規則を導きだすことができる。

導入

リストの例

以下のような多相関数rに対して

r :: [a] -> [a]

以下の規則が成り立つ。

map f . r == r . map f

直観的な説明

関数rはすべての型を要素とするリストに対して定義されるので、 要素自体を処理することはできない。 リストの並びかたを変化させることだけしかできない。 各要素への関数の適用と要素の並びの変更はどちらを先にしても結果は同じとなる。
