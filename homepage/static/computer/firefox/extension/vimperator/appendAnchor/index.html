appendAnchorプラグイン

目的

anekosさんのappendAnchorプラグインを試してみる。

プラグインの導入の方法

天井さんのプラグインの導入についての記事を参照する。 どうやら、以下のようにすれば良いようだ。

  1. ~/.vimperatorにpluginディレクトリを作成
  2. 導入したい.jsファイルをそこに置く
  3. .vimperatorrcに":source"コマンドを書く

実際にやってみる

以下の作業を実際にやってみることにする

  1. mkdir ~/.vimperator/plugin
  2. ここから.jsファイルを取得する
  3. 上で取得したファイルをpluginディレクトリに移動させる
  4. まずはオンザフライで:source appendAnchorをやってみる
  5. リンクじゃないURIを用意する
  6. ヒントモードを試してみる
  7. .vimperatorrcに:source appendAnchorを追加する

テスト用のURI1: https://www.google.co.jp/

テスト用のURI2: /computer/vimperator

結果としては':source appendAnchor'はうまくいかない。 また、.vimperatorrcに上記を書く必要はなく、 pluginディレクトリ以下に.jsファイル置いて、 firefoxを再起動すれば良いようだ。

実際に必要な手順

  1. mkdir ~/.vimperator/plugin
  2. ここから.jsファイルを取得する
  3. 上で取得したファイルをpluginディレクトリに移動させる
  4. firefoxを再起動する
  5. :ancコマンドを打つと裸のURIにアンカーがつく
