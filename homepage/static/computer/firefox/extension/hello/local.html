ローカルに拡張機能をインストール

(要見直し)

このページの内容に沿って拡張機能を作成し、 ローカルにインストールしてみよう。 内容が古くなっているところがあるので、そこらへんを修正しながらやっていく。 このページの内容はfirefoxの24.8.0で確認している。

install.rdf

helloworldディレクトリを作成し、そこにinstall.rdfを置く。 このページのinstall.rdfを参考に以下の修正をした。 バージョンに関してはここを参照。 また、ここではhelloworld@skami.iocikun.jpだが、 skami.iocikun.jpの部分は各自自分の持っているドメイン名に変更すること。

  • 英語版の説明は削除
  • helloworld@yourname.netをhelloworld@skami.iocikun.jpに
  • minVersionを3.0.1から24.8.0に
  • maxVersionを3.*から24.*に

修正後のinstall.rdf

XULファイル

このページを参考にする。

適当なディレクトリ(/home/hoge/とする)以下にhelloworldディレクトリを作成する。 helloworldディレクトリ以下にcontentディレクトリを作成し、helloworld.xulを置く。

Firefox 4からはステータスバーが変更され、常時表示されるものではなくなった。 代わりにアドオンバーが用意された。 よって以下の変更を行った。

  • <statusbar id="status-bar">を<toolbar id="addon-bar">に

chrome.manifest

helloworldディレクトリにchrome.manifestを置く。 これはとくに変更は不要だった。

helloworld@skami.iocikun.jp

上にも書いたがskami.iocikun.jpは各自 自分の持っているドメイン名に変更する必要がある。 プロファイルディレクトリにextensionsディレクトリを作成し、 helloworld@skami.iocikun.jpをそこに置く。 このファイルには今まで作ってきたファイルを置いてあるディレクトリ名を書いておく。 /home/hoge/helloworld/のような一行を書いおくということだ。

ローカルなインストールが完了

% firefox -no-remote -P dev
で確認してみよう。
