はじめに

SQLiteはデータベースに単一のファイルを使うので使いやすい、ゆるふわなSQLだ。 ゆるふわとは言っても中規模なデータベースにおけるベンチマークで 断トツなスコアを記録するなどなかなかの実力の持ち主だ。 実用的なプロジェクトでもとくに理由がなければSQLiteを使うのもひとつの選択肢だ。
