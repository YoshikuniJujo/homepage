Hubot

(工事中 0%)

インストール

% npm install -g hubot hubot-slack yo generator-hubot coffee-script

ひな形の作成

% mkdir hellobot
% cd hellobot
% yo hubot

scripts

scriptsディレクトリ下にスクリプトを作成する。

% vi scripts/helloworld.coffee
module.exports = (robot) ->
robot.hear /HELLO$/i, (msg) ->
msg.send "world"

動作確認

% ./bin/hubot

エラーメッセージが出るがエンターを押す。

Hubot> Hubot hello
Hubot> world
exit

Slackと連携する

SlackのIntegrationページでHubotとの連携を設定する。 HUBOT_SLACK_TOKENをメモする。

% HUBOT_SLACK_TOKEN=xxxxxxxxxxxxxx HUBOT_SLACK_TEAM=xxxx HUBOT_SLACK_BOTNAME=hubot bin/hubot --adapter slack
