JavaScript Object Notation (JSON)は...

JSONは4つのプリミティブな型(文字列、数値、真偽値、Null値)と 2つの構成的な型(オブジェクト、配列)を表せる。

文字列は0以上のユニコード文字の並びである。 このcitationは特定のリリースを指すのではなくユニコードの最新版を参照する。 将来のユニコードの仕様の変化がJSONの文法に大きな影響を与えることはないだろう。

オブジェクトは0以上の名前/値のペアの順序なしの集合である。 名前は文字列であり、値は文字列、数値、真偽値、Null値、オブジェクト、配列だ。

配列は0以上の値の順序つきの並びだ。

「オブジェクト」や「配列」という用語はJavaScriptの慣用からだ。

JSONの設計のねらいはミニマルであること、ポータブルであること、テキストであること、 そしてJavaScriptのサブセットであることだ。
