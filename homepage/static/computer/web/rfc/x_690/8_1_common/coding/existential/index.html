X.690: ASN.1のBER方式の共通部分の存在型を利用したコーディング

動機

ASN.1は拡張可能であり、もちろんBER方式も拡張可能だ。 よってデコード結果となる型を追加できるようにしておきたい。 存在型を使用することでそれが可能になる。

参照: 存在型について

現在の状況

いろいろと設計の試行錯誤をしている。 最終的な設計が決定した段階で全体をわかりやすい道筋に編集し直す予定。

コーディング

  • データ型の決定
  • デコード関数
  • デコード関数の拡張: Ruleの追加
  • タグやデータ長のエンコード関数
  • エンコードのためのデータ型の見直し
  • デコードルールをクラス関数にする
  • モジュールに分ける (工事中 10%)
  • エンコード関数を値自体に持たせる (中断 1%)
  • エンコードとデコードとを分ける (工事中 0%)
  • BER形式でのエンコード
  • BER形式のデコーダを独立させる
