X.690: ASN.1のBER方式の共通部分のデータ型

Asn1はタグ(Asn1Tag)と内容(Asn1またはByteString)から成る。 Asn1Tagはタグのクラス(TagClass)とデータのクラス(DataClass)とタグの番号から成る。 

data Asn1 = Asn1 Asn1Tag Asn1Data
	deriving Show
data Asn1Tag = Asn1Tag TagClass DataClass Integer
	deriving Show
data TagClass =
	Universal | Application | ContextSpecific | Private
	deriving Show
data DataClass = Primitive | Constructed
	deriving Show
data Asn1Data
	= Asn1DataAsn1 [Asn1]
	| Asn1DataRaw ByteString
	deriving Show
