RFC 2083: メモ

ヘッダ

"\x89PNG\r\n\SUB\n"

  • 最初の2バイトでファイルの種類を特定するシステムへの対応
  • 最初の1バイトがASCII文字ではない
    • テキストファイルをPNGファイルとして処理することを阻止
    • ビット7のクリアを検出可能
  • \r\nと\nが含まれている
    • テキストモードでの転送を検出可能
  • control-Z(\SUB)が含まれる
    • MS-DOSでのファイルの「表示」を阻止

チャンクの構造

  • Length: 4バイト
  • Chunk Type: 4バイト
  • Chunk Data
  • CRC: 4バイト
