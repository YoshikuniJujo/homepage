Firefox OSの構成

以下の3つの層から成る。

  • Gaia
  • Gecko
  • Gonk

下から説明する。

Gonk

カーネル層。 Android用に修正されたLinuxを使用。 Android 4.0からJava仮想マシンのDalvicやWebkit等を削除している。 デバイスドライバもAndroidのものを使っている。

Gecko

HTML、CSS、Javascript等を理解しWeb APIを用意する層。 Webアプリはこの上で走る。 Firefoxブラウザで使用されているのと同じWebエンジン。 システムアプリも含め、すべてのアプリがWebアプリと同様の技術で作られる。

Gaia

ホームアプリや電話、メール等の基本的なアプリから成る。 基本的なユーザインターフェースの層。 HTML、CSS、JavascriptなどWebアプリと同じ技術で作られる。

Firefox OSにおけるアプリとは

他のスマートフォン向けのOSと同様にWebアプリとネイティブアプリとがあるが、 その二者の違いは小さい。 他のOSではネイティブアプリはiPhoneならObjective-Cを、 AndroidならJavaを使用している。 Firefox OSではWebアプリと同じ技術を使用している。 また、Androidではアプリの開発に特殊な環境を必要とするが、 Firefox OSではWebアプリを開発するのと同様に標準的な環境で開発が行える。

Webアプリ

普通にHTML、CSS、Javascript等によってサイトを構成すれば良い。 マニフェストファイルを作成しインストール可能なアプリとすることができる。 ただし1ドメインにつき1つのアプリケーションしか公開できない。

パッケージ化されたアプリ

マニフェストファイルを含むファイル群をzip圧縮する。 インストール用のマニフェストを用意し、Javascriptでインストール機能を用意する。
